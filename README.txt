# sort
Кодировка windows-1251/CP1251
Предполагается ввод полного пути к файлу
Пример: D://Bunin/CN/winpcap/bubble.txt
Необходимо изменить путь к библиотеке в файле *.pro,
иначе программа не будет работать.
В версии с окошками, если число пакетов выставить, как "full",
то может показаться, что программа зависла, но если открыть папку 
с программой, то видно, как записываются файлы.
Графический интерфейс подвисает, и lable'ы начинают отображаться
только после завершения программы, пока не разобрался, как пофиксить
Хотя изначально была задумка увеличения размера окошка и отображения
дополнительной информации по мере выполнения завершения сортировок,
окошко то увеличивалось, но надписи не появлялись, как уже писал - 
пока не знаю, как пофиксить
